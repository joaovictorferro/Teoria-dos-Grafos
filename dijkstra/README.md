Para compilar o código:

Na pasta do arquivo, run:

> g++ dijkstra.cpp

Para executar com a entrada:

No Windows:

> .\a.exe < in.txt

No Linux:
> ./a.out < in.txt

Sobre a entrada:
5 6
1 2 4
1 3 3
4 3 6
4 5 2
2 4 1
3 5 5

Primeira linha:
Número de vértices N (1-N) e número de arestas M
Cada M linha:
A B C
Vértice A para B e B para A, C = custo.

Resultado esperado:
7
